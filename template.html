Ruby on Rails

The goal of this project is to build a web based application that provides a platform for users of the application to search for restaurants by name, address, city or zip code and submit/view reviews on the restaurant. Users of the application sign up for an account on the website and login before submitting reviews (preventing anonymous users from spamming the application).

PHP

The goal of this project is to build a web based application that provides a platform for users of the application to post ads about any musical instruments that they want to rent/sell. All members/users of the site can view the ads, save ads as favorites to revisit later. This website does not provide messaging mechanism between members to kick off the purchase/rent. That is assumed to be an offline activity by the members. The aim is to provide a platform to know about sellers and their products and obtain the contact information of the sellers..
